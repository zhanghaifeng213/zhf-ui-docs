    # 介绍

    # 写在前面

    这是本人在学习 Vue 的过程中制作的一个基于 Vue2.0 的 UI 组件库，用意是加深对 Vue 的各种概念的理解，提高自己的水平。

    在完成此 UI 组件库的过程中，从第一个组件，发布到 npm，再逐渐完善增加其他组件，学习到很多东西。

    本组件库仅供学习交流, 欢迎讨论交流。

    # 项目特点

    • 深入使用了 Vue 的各种特性。

    • 组件样式与交互参考了微信小程序 Lin UI 组件库,Lin UI。
